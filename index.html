Setup Instructions

Configure OpenClaw and OpenCode (Opencode) to use this optional router/dashboard layer.

OpenClaw

Automatic

  1. Run model-foundry onboard (or modelrelay onboard for legacy compatibility).
  2. When asked Auto-configure OpenClaw now? [Y/n], answer Y.
  3. Start the router with model-foundry (or modelrelay for legacy compatibility).

Manual

Merge this into ~/.openclaw/openclaw.json: 
{
  "models": {
    "providers": {
      "modelfoundry": {
        "baseUrl": "http://127.0.0.1:7352/v1",
        "api": "openai-completions",
        "apiKey": "no-key",
        "models": [
          { "id": "auto-fastest", "name": "Auto Fastest" }
        ]
      }
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "openai/gpt-5.4"
      },
      "models": {
        "modelfoundry/auto-fastest": {}
      }
    }
  }
}

OpenCode (Opencode)

Automatic

  1. Run model-foundry onboard (or modelrelay onboard for legacy compatibility).
  2. When asked Auto-configure OpenCode now? [Y/n], answer Y.
  3. Start the router with model-foundry (or modelrelay for legacy compatibility).

Manual

Put this in ~/.config/opencode/opencode.json: 
{
  "$schema": "https://opencode.ai/config.json",
  "provider": {
    "router": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "modelfoundry",
      "options": {
        "baseURL": "http://127.0.0.1:7352/v1",
        "apiKey": "dummy-key"
      },
      "models": {
        "auto-fastest": {
          "name": "Auto Fastest"
        }
      }
    }
  },
  "model": "router/auto-fastest"
}
